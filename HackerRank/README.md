# Arrays

## Left Rotation
Given an array of `n` integers and a number `d`, perform `d` left rotations on the array. Then print the updated array as a single line of space-seperated integers.

Example:
```
Input:  arr = [1,2,3,4,5], d = 2
Output: arr = [3,4,5,1,2]
```
